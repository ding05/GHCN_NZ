# GHCN NZ

The code generates an adjacency matrix for the Global Historical Climatology Network (GHCN) in New Zealand and visualizes the network.

## Datset

[GHCND Stations](https://www.ncdc.noaa.gov/ghcnd-data-access)

## Tools

[pandas](https://pandas.pydata.org)

[GHCN Adjacency Matrix](https://github.com/ding05/GHCN_Adjacency_Matrix)

[tikz-network](https://ctan.org/tex-archive/graphics/pgf/contrib/tikz-network)

[Google Map](https://www.google.com/maps)
